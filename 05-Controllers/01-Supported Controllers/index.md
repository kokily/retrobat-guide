# 컨트롤러 지원

Windows 운영 체제와 호환되는 모든 주요 컨트롤러는 Retrobat에서 작동해야 합니다.

그러나 Windows에서 컨트롤러 드라이버를 관리하므로 최상의 호환성을 위해 X-Input 컨트롤러를 사용하는 것이 좋습니다.

Wiki의 하위 페이지에는 팀에서 테스트한 컨트롤러에 대한 정보가 포함되어 있습니다.
